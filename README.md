# My GitHub Pages Site

Source for my personal site, hosted with [GitHub Pages](https://pages.github.com/).

🔗 **Live site:** https://aksucode.github.io

For local testing serve the files from http server: 

Windows:

Open command prompt

Navigate to project root

run: python -m http.server 8000

Open browser at: http://localhost:8000/index.html


Libs Install:

Three: https://cdn.jsdelivr.net/npm/three@0.185.1/build/three.module.js

Core: https://cdn.jsdelivr.net/npm/three@0.185.1/build/three.core.js

List of Addons: https://cdn.jsdelivr.net/npm/three@0.185.1/examples/jsm/
